# RestoHub Backend

## Requisitos

- Python 3.11
- Docker
- Docker Compose
- PostgreSQL
- RabbitMQ

## Instalación

1. Clonar el repositorio
2. Abrir la terminal en la carpeta del repositorio
3. Ejecutar los siguientes comandos:

```bash
docker compose up -d
```

tumbar docker sin borrar los contenedores

```bash
docker compose down
```

volver a arrancar los contenedores

```bash
docker compose up
```

Python + Django

crear un nuevo entorno virtual

```bash
python -m venv env
```

correr el script de activacion

```bash
source env/bin/activate
env\Scripts\activate
```

instalar las dependencias

```bash
pip install -r requirements.txt
```
